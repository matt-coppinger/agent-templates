# Widget Pro

**Last updated:** 2026-02-15
**Category:** products

## Overview

The Widget Pro is our flagship productivity tool designed for professional users. It combines advanced features with an intuitive interface to streamline daily workflows.

## Key Features

- **Smart Dashboard** — customisable workspace with drag-and-drop widgets
- **Team Collaboration** — real-time co-editing with up to 25 team members
- **Automated Reports** — schedule daily, weekly, or monthly reports
- **API Access** — RESTful API for custom integrations (Business plan and above)
- **Mobile App** — full-featured iOS and Android companion apps

## System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge — latest two versions)
- Mobile: iOS 16+ or Android 13+
- Internet connection (minimum 2 Mbps recommended)

The Widget Pro comes with a 12-month warranty covering manufacturing defects and software bugs. Premium support (24/7 live chat) is included with Business and Enterprise plans. Starter and Professional plans include email support with a 24-hour response time.
